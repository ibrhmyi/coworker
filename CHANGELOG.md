# Changelog

## 0.1.0-alpha.1 (2026-04-17)

Initial alpha release.

### Features
- 6 MCP tools: submit_task, wait_for_task, get_result, iterate_task, list_tasks, get_project_state
- Async parallel task execution
- Session resumption for cheap iteration
- Auto-verification with configurable test/build commands
- Heuristic and LLM-powered summaries
- Quick and named Cloudflare tunnels
- Auto-managed project state (STATUS.md, CONTEXT.md, DECISIONS.md)
- One-command setup with cloudflared auto-download
- 8 CLI commands
- 77 tests
